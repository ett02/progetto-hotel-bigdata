# Hotel Big Data Analysis

## Come avviare l'applicazione

### Metodo 1: Script Automatico (Consigliato)
Fai doppio click sul file **`run_app.bat`** presente in questa cartella.

### Metodo 2: Manuale da Terminale
1.  Apri un terminale (Command Prompt o PowerShell).
2.  Naviga nella cartella del progetto:
    ```bash
    cd f:\prog_big_1\frontend
    ```
3.  Esegui il comando:
    ```bash
    streamlit run app.py
    ```
    *Se `streamlit` non viene riconosciuto, prova:*
    ```bash
    python -m streamlit run app.py
    ```
